Simple Form






<form></form>
 Esse elemento é usado para criar um formulário para entrada de usuários em HTML

<input></input>
 Esse elemento é o elemento de formulário mais utilizado.

Ele pode ser exibido de várias maneiras, depende do type atribuido.

<label></label>
 Define um rótulo para muitos elementos de formulário.

Esse elemento ajuda os usuários que tem dificuldade em clicar em regiões muito pequenas.

Importante:
O atributo for da tag <label> deve ser igual ao atributo id do elemento <input> para uni-los.
