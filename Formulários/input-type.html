<input type="text">
 Exibe um campo de entrada de texto de linha única.




<input type="radio">
 Exibe um botão de opção (selecionar uma das muitas opções).




<input type="checkbox">
 Exibe uma caixa de seleção (para selecionar zero ou mais opção).
A tradução de checkbox é caixa de seleção.





<input type="submit">
 Exibe um botão de envio (para enviar formulário)
A tradução de submit é enviar.

<input type="button">
 Exibe um botão clicável.
