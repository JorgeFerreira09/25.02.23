Input Restrictions

Range Field



<input type="range">

Define um controle para inserir um número cujo valor exato não é importante (como um controle deslizante).

O intervalo padrão é de 0 a 100.

No entanto, você pode definir restrições sobre quais números são aceitos com os atributos abaixo.

max - especifica o valor máximo permitido
min - especifica o valor mínimo permitido
passo - especifica os intervalos de número legal
valor - Especifica o valor padrão
Dica: Sempre adicione a
